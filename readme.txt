Lab 6

У прикладі в файлі tim.py користувач вводить оптимістичну, найбільш ймовірну та песимістичну оцінки для двох завдань. Після введення всіх оцінок програма обчислює середнє значення 'e' та стандартне відхилення 'sd' для кожного завдання, а потім обчислює загальне середнє значення та стандартне відхилення для проекту. Нарешті, виводиться 95% інтервал довіри для проекту.

У вище наведеному прикладі інтервал довіри складає від 4.57 до 7.43 балів. Значить, з 95% впевненістю можна очікувати, що сумарна оцінка проекту буде між цими двома значеннями.
